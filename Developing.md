# How to build Superslash Docker Registry Center

This file contains tips to help you take (and understand) your first steps in Superslash Docker Registry Center development.

## Clone and install the repository

```bash
git clone https://github.com/Joxit/docker-registry-ui.git
cd docker-registry-ui
npm install
```

## Run the local server

```bash
npm start
```

Open your browser <http://localhost:8000> you can configure your options by updating the `src/index.html` file.
